# GUI Calculator (Java)

This is a simple GUI based calculator built using Java.  
The calculator can perform basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features

- Simple and clean graphical user interface
- Performs basic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Built using Java GUI libraries
- Easy to understand code for beginners

## Technologies Used

- Java
- Swing / AWT (for GUI)

## How to Run the Project

1. Clone the repository

git clone https://github.com/preetipandit-dev/GUI-Calculator.git

2. Open the project in any Java IDE (VS Code, IntelliJ, Eclipse)

3. Compile the program

javac Calculator.java

4. Run the program

java Calculator
